Создай веб-приложение Gamified Life OS в стиле игры Persona 5 Royal.

Это приложение должно быть инструментом ежедневной продуктивности и организации жизни, который сочетает:

gamified task manager

систему целей

календарь задач

focus-таймер

систему XP и статистики

раздел заметок по трейдингу

раздел финансовых новостей

Приложение должно быть полностью на русском языке.

Главная цель приложения —
помогать пользователю выполнять ежедневные задачи и двигаться к недельным, месячным и годовым целям через понятную систему XP.

1. VISUAL STYLE

Интерфейс должен быть вдохновлён стилем игры Persona 5.

Основные цвета:

чёрный

красный

белый

Стиль интерфейса:

резкие геометрические формы

comic-style UI

плавные анимации

минималистичный дизайн

динамичные переходы

2. ГЛАВНАЯ СТРАНИЦА (MAIN HUB)

Главная страница — это центральный хаб приложения.

В центре должен находиться анимированный персонаж в стиле lofi productivity.

Он должен менять своё состояние в зависимости от времени дня (по времени Берлина UTC+1).

Состояния:

00:00 – 06:00
персонаж спит

06:00 – 09:00
персонаж начинает день (утренние действия)

09:00 – 14:00
персонаж работает

14:00 – 21:00
персонаж отдыхает

21:00 – 00:00
спокойный вечер

Все переходы должны быть плавными анимациями.

3. ОТОБРАЖЕНИЕ ВРЕМЕНИ И СЕССИИ РЫНКА

Приложение должно в реальном времени показывать:

текущее время пользователя

текущую торговую сессию

Часовой пояс:

UTC+1 (Берлин)

Сессии:

Asia — 03:00–08:00
Frankfurt — 08:00–09:00
London — 09:00–14:00
New York — 14:00–17:00

Индикатор сессии должен:

автоматически обновляться

показывать правильную активную сессию

НЕ иметь задержки в 1 час

4. СФЕРЫ ЖИЗНИ (LIFE AREAS)

Body

5. СИСТЕМА ЗАДАЧ

В системе должно существовать 3 типа задач.

TYPE 1 — РУТИНА

Это задачи, которые повторяются каждый день.

Примеры:

спорт
чтение
медитация
подготовка к торговле

Особенности:

фиксированные XP

повторяются ежедневно

могут редактироваться

создаются через шаблон рутины

TYPE 2 — ЗАДАЧИ НА СЕГОДНЯ

Это задачи, которые добавляются только на текущий день.

Особенности:

не переносятся на следующий день

имеют уровень сложности

дают XP

TYPE 3 — ЦЕЛЕВЫЕ ЗАДАЧИ

Задачи, которые привязаны к:

недельным целям

месячным целям

годовым целям

6. СТРУКТУРА ВКЛАДКИ ЗАДАЧ

Вкладка задач должна содержать две вкладки:

1️⃣ РУТИНА
(шаблон задач, которые повторяются ежедневно)

2️⃣ ЗАДАЧИ НА СЕГОДНЯ
(задачи только на текущий день)

При этом:

в главном хабе должен отображаться единый список задач, который включает:

задачи рутины

задачи сегодняшнего дня

Задачи из рутины должны иметь тег "Рутина".

7. ШАБЛОН РУТИНЫ

Пользователь должен иметь возможность:

создать шаблон ежедневной рутины.

После нажатия кнопки "Загрузить рутину":

задачи из шаблона должны:

добавляться в список задач на сегодня

НЕ дублироваться в шаблоне рутины

НЕ создавать копии в списке шаблона

Это важно для предотвращения предыдущей ошибки.

8. УДАЛЕНИЕ ЗАДАЧ

Должна быть кнопка "Очистить список задач".

При нажатии должно появляться подтверждение.

Без подтверждения удаление невозможно.

9. СИСТЕМА СТРЕАКОВ (STREAK)

Streak засчитывается только если выполнены ВСЕ задачи дня.

Это означает:

выполнена вся рутина

выполнены все задачи на сегодня

Если хотя бы одна задача не выполнена:

streak сбрасывается.

Streak должен:

начинаться со второго дня

не учитываться в первый день

Если задача была:

добавлена → удалена

она не должна влиять на streak.

10. СИСТЕМА ЦЕЛЕЙ

Цели имеют 3 уровня:

YEAR GOALS
MONTH GOALS
WEEK GOALS

Связь должна быть иерархической:

Year goal
→ содержит Month goals

Month goal
→ содержит Week goals

Week goal
→ содержит задачи

Пользователь должен иметь возможность:

привязывать конкретные задачи к конкретной цели.

11. XP СИСТЕМА (ПЕРЕРАБОТАННАЯ)

XP разделены на 3 независимых канала:

Routine XP
Task XP
Goal XP

XP не смешиваются между каналами.

ROUTINE XP

Каждая рутина имеет фиксированное значение.

Workout → 10 XP
Reading → 10 XP
Meditation → 5 XP
Journal → 5 XP

Максимум:

50 XP в день.

DAILY TASK XP

3 уровня сложности.

Low → 10 XP
Medium → 25 XP
High → 50 XP

GOAL TASK XP

Week Goal → 100 XP
Month Goal → 250 XP
Year Goal → 1000 XP

12. ФОКУС ТАЙМЕР

Режимы:

Pomodoro (25 мин)
Deep Work (60–90 мин)
Custom timer

XP:

25 минут → 5 XP
60 минут → 15 XP
90 минут → 25 XP
